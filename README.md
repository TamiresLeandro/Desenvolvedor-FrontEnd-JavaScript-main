# 📚 Estudos de JavaScript: Desenvolvedor Web Front-end

Este repositório contém o progresso dos meus estudos baseados no livro **"Desenvolvedor web front-end: JavaScript"** da Editora Senac São Paulo. O objetivo é documentar a evolução no aprendizado da linguagem, desde os fundamentos até a manipulação avançada do DOM e integração com bibliotecas.

---

## 🛠️ Tecnologias e Conceitos Explorados
* **Linguagem:** JavaScript (ES6+)
* **Estruturas:** Variáveis, Decisão (if/else), Repetição (loops).
* **DOM:** Manipulação dinâmica de elementos HTML e estilos CSS.
* **Bibliotecas:** Introdução ao jQuery.
* **Projetos:** Banners animados, galerias lightbox e validação de formulários.

---

### 🔹 [Capítulo 01 - Primeiros Passos](./Capitulo01/)
